Socket RPG V2

掲載日: 1999/04/11, 更新日: 2000/10/02

［お知らせ］

Socket RPGの新規バージョン (V2.00) を公開しました。 Vectorさんよりダウンロード可能です。

V2.00の主な変更点は以下になります。

  • ライセンス・キーのロジックをはずします。(FreeSoftとします。）
  • APIを追加します。（SKWAIT,SKSNDF,SKSNDL)など
  • いつかの API に関してバグ修正やパフォーマンス改善しています。

Socket RPGは、RPG で TCP/IP対応の通信プログラムの作成を可能とします。

＜Socket RPGを使用したコーディング例＞


.....省略.......

  C*ソケットオープン--------------------------------------*
  C                     CALL 'SKOPEN'
  C                     PARM           SD      4
  C                     PARM           RC      1
  C*ソケットコネクト--------------------------------------*
  C                     CALL 'SKCONN'
  C                     PARM           SD      4
  C                     PARM           RC      1
  C                     PARM           IP
  C                     PARM '30001'   PORT    5
  C*ソケットデータ送信------------------------------------*
  C                     CALL 'SKSEND'
  C                     PARM           SD      4
  C                     PARM           RC      1
  C                     PARM '256'     BUFLEN  4
  C                     PARM           BUFOUT256
  C*ソケットデータ受信------------------------------------*
  C                     CALL 'SKREAD'
  C                     PARM           SD      4
  C                     PARM           RC      1
  C                     PARM '256'     BUFLEN  4
  C                     PARM           BUFIN 256
  C*------------------------------------------------------*
.....省略.......

  • RPG 数行のコーディングで TCPIP 通信が利用可能
  • TCP, UDP, 両タイプの通信をサポート
  • 文字コード変換も可能
  • デーモンタイプのアプリケーションも構築可能
