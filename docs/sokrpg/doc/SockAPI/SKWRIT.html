SKWRIT

構文： 
     C*ソケットデータ送信（もらったデータの折り返し）--------*
     C                     CALL 'SKWRIT'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM '256'     BUFLEN  4
     C                     PARM           BUF
     C*------------------------------------------------------*
解説：
ソケットにデータを送信する。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。
BUFLEN　(入力)
    送信するデータの長さ
BUF　(入力)
    実際に送信するデータ、BUFLENで指定した長さ以上を用意すること

実行されるソケット関数：
write()
