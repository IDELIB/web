SKSNDC

構文： 
     C* DATA送信 --------------------------------------------*
     C                     CALL 'SKSNDC'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM '5026'    FCCSID  5
     C                     PARM '932'     TCCSID  5
     C                     PARM 'Y'       SOSISP  1              Y/N 
     C                     PARM           TERMCH  2
     C                     PARM           BUFLEN  4
     C                     PARM           BUF
     C*------------------------------------------------------*
解説：
FCCSIDからTCCSIDにデータを変換後に、ソケットにデータを送信する。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。
FCCSID　(入力)
    変換元のCCSIDをセットする。
TCCSID　(入力)
    変換先のCCSIDをセットする。
SOSISP　(入力)
    シフトコードをブランク１文字として変換するかどうか。
    'Y'ブランクにする。
    'N'ブランクにしない。
TERMCH　(入力)
    データの終わりに付けたい改行コードを設定する
    CRLF(0x0d0a)
    LF(0x0a00)
    NULLL(0x0000) など
BUFLEN　(入力)
    送信するデータの長さ
BUF　(入力)
    実際に送信するデータ、BUFLENで指定した長さ以上を用意すること

実行されるソケット関数：
send()
