SKGETS

構文： 
     C*ローカルポート検索------------------------------------*
     C                     CALL 'SKGETS'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM           IP     15
     C                     PARM           PORT    5
     C*------------------------------------------------------*
解説：
ソケットのローカルポートを検索する。

パラメーター：
SD　(入力)
    検索するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。
IP　(出力)
    このソケットが使用している構内のIPアドレスの文字表現
PORT　(出力)
    このソケットが使用している構内PORT番号

実行されるソケット関数：
getsockname()
