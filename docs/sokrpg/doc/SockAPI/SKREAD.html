SKREAD

構文： 
     C*ソケットデータ受信------------------------------------*
     C                     CALL 'SKREAD'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM '256'     BUFLEN  4
     C                     PARM           BUF
     C*------------------------------------------------------*
解説：
ソケットをからデータを受信します。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、および、通信相手がソケットをクローズした場合に'E'に設定される。
BUFLEN　(入出力)
    受信するバッファーの長さ（９９９９まで可能）、戻りは、実際に受信したデータの長さ。 ゼロの戻り値は通信相手がソケットをクローズしたことを示す。
BUF　(出力)
    実際にデータを受信するバッファー。少なくともBUFLENで指定した長さは必要。

実行されるソケット関数：
read()
