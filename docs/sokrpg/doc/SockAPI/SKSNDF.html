SKSNDF

構文： 
     C* ファイル送信 --------------------------------------------*
     C                     CALL 'SKSNDF'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM           PFILE  10
     C*------------------------------------------------------*
解説：
パラメーターで渡されたファイルをオープンし、ファイルの全てのレコードをソケットに送信する

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。
PFILE　(入力)
    送信するデータの入っているファイル名を左詰めに設定する。

    ファイルはライブラリーリストに存在していること。

実行されるソケット関数：
send()
