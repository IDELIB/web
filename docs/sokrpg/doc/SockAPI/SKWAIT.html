SKWAIT

構文： 
     C*受信待機（タイムアウト２０秒）----------------------*
     C                     CALL 'SKWAIT'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM '00000020'TIMOUT  8
     C*------------------------------------------------------*
解説：
ソケットにデータが着信するのを待つ。事前にノンブロックの指定をする必要なし

SKACPT,SKRECVの前に実行可能。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。

    タイムアウトした場合、'T'に設定される。

TIMOUT　(入力)
    タイムアウトするまでの秒数を指定する。文字タイプです

実行されるソケット関数：
select()
fcntl()
