SKSNDS

構文： 
     C* DATA送信 --------------------------------------------*
     C                     CALL 'SKSNDS'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM 'Y'       SOSISP  1              Y/N 
     C                     PARM           BUFLEN  4
     C                     PARM           BUF
     C*------------------------------------------------------*
解説：
シフトJISにデータを変換後に、ソケットにデータを送信する。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。
SOSISP　(入力)
    シフトコードをブランク１文字として変換するかどうか。
    'Y'ブランクにする。
    'N'ブランクにしない。
BUFLEN　(入力)
    送信するデータの長さ
BUF　(入力)
    実際に送信するデータ、BUFLENで指定した長さ以上を用意すること

実行されるソケット関数：
send()
