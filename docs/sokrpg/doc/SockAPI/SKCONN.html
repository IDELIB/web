SKCONN

構文： 
     C*ソケットコネクト--------------------------------------*
     C                     CALL 'SKCONN'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM           IP     15
     C                     PARM           PORT    5
     C*------------------------------------------------------*
解説：
相手先ソケットに接続する。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。
IP　(入力)
    接続先のIPアドレス、文字形式でxxx.xxx.xxx.xxxと指定する
PORT　(入力)
    接続先のPORT番号、文字形式で指定する。右づめでも左づめでも可

実行されるソケット関数：
connect()
