SKGETH

構文： 
     C*ホスト名検索------------------------------------------*
     C                     CALL 'SKGETH'
     C                     PARM           HOSTNM128
     C                     PARM           RC      1
     C                     PARM           IPSTR  15
     C                     PARM           IPHOST128
     C*------------------------------------------------------*
解説：
ホスト名またはＩＰアドレスを検索する

パラメーター：
HOSTNM　(入力)
    検索するホスト名、または、IPアドレス
RC　(出力)
    エラーが発生した場合、'E'に設定される。
IPSTR　(出力)
    検索されたIPアドレス
IPHOST　(出力)
    検索されたホスト名

実行されるソケット関数：
gethostbyname
gethostbyaddr
