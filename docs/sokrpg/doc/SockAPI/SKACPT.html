SKACPT

構文： 
     C*ソケットアクセプト------------------------------------*
     C                     CALL 'SKACPT'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM           ND      4
     C*------------------------------------------------------*
解説：
接続要求を受けたソケットの接続要求を受け入れます。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。
ND　(出力)
    接続に新たに割り振られたソケットディスクリプターの番号

実行されるソケット関数：
accept()
