SKGIVE

構文： 
     C* 他のＪＯＢにソケットをあたえる ----------------------*    
     C                     CALL 'SKGIVE'                            
     C                     PARM           SD      4                 
     C                     PARM           RC      1                 
     C                     PARM 'SOKBCH  'APLID   8
     C*------------------------------------------------------*
解説：
SKTAKEを利用してソケットディスクリプター獲得を待機しているＪＯＢに、ソケットを渡します

パラメーター：
SD　(入力)
    渡そうとするソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合に'E'に設定される。
    SKTAKEで待機しているJOBがいなかった場合に、'N'に設定される。
APLID　(入力)
    ソケットディスクリプターの受渡しの可能な複数JOBを識別するために付ける任意の文字列

実行される関数：
givedescriptor()
