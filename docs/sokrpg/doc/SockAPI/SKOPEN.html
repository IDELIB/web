SKOPEN

構文： 
     C*ソケットオープン--------------------------------------*
     C                     CALL 'SKOPEN'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C*------------------------------------------------------*
解説：
TCPソケット(ソケット・ストリーム)を作成します

パラメーター：
SD　(出力)
    割り振られたソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。

実行されるソケット関数：
socket(AF_INET, SOCK_STREAM, 0)
setsockopt(sd, IPPROTO_TCP,TCP_NODELAY,(char *)&on,sizeof(on))
setsockopt(sd, SOL_SOCKET,SO_REUSEADDR,(char *)&on,sizeof(on))
