SKUDPS

構文： 
     C* UDPソケットデータ送信--------------------------------*
     C                     CALL 'SKUDPS'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM '256'     BUFLEN  4
     C                     PARM           BUF
     C                     PARM           IP     15
     C                     PARM           PORT    5
     C*------------------------------------------------------*
解説：
ソケットをからデータを送信します。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合に'E'に設定される。
BUFLEN　(入力)
    受信するバッファーの長さ（９９９９まで可能）。
BUF　(入力)
    実際にデータを送信するバッファー。少なくともBUFLENで指定した長さは必要。
IP　(入力)
    データ送信先のIPアドレスの文字表現(xxx.xxx.xxx.xxx)
PORT　(入力)
    データ送信先のPORT番号

実行されるソケット関数：
sendto()
