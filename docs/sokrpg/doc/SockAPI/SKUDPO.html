SKUDPO

構文： 
     C*ＵＤＰソケットオープン--------------------------------*
     C                     CALL 'SKUDPO'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C*------------------------------------------------------*
解説：
UDPソケット(ソケット・ダイアグラム)を作成します。

パラメーター：
SD　(出力)
    割り振られたソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。

実行されるソケット関数：
socket(AF_INET, SOCK_DGRAM, 0)
setsockopt(sd, SOL_SOCKET,SO_REUSEADDR,(char *)&on,sizeof(on))
