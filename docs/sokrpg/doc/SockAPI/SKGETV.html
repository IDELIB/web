SKGETV

構文： 
     C*サービス名検索----------------------------------------*
     C                     CALL 'SKGETV'
     C                     PARM           SERVNM 32
     C                     PARM           PROTNM 32
     C                     PARM           RC      1
     C                     PARM           PORT    5
     C                     PARM           SERV   32
     C                     PARM           PROT   32
     C*------------------------------------------------------*
解説：
サービス名を検索します

パラメーター：
SERVNM　(入力)
    検索するサービス名、または、PORT番号
PROTNM　(入力)
    検索するプロトコル名(UDP,TCPなど) 省略可能
RC　(出力)
    エラーが発生した場合、'E'に設定される。
PORT　(出力)
    検索されたPORT番号
SERV　(出力)
    検索されたサービス名
PROT　(出力)
    検索されたプロトコル名

実行されるソケット関数：
getservbyname()
getservbyport()
