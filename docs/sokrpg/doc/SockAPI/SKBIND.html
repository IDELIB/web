SKBIND

構文： 
     C*ソケットバインド--------------------------------------*
     C                     CALL 'SKBIND'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM           IP     15
     C                     PARM           PORT    5
     C*------------------------------------------------------*
解説：
作成したソケットにＩＰおよびＰＯＲＴを割り当てます。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。
IP　(入力)
    このソケットに割り当てるインターフェースのIPアドレス。インターフェースを特定しない場合は"ANY"を設定する
PORT　(入力)
    このソケットに割り当てるPORT番号

実行されるソケット関数：
bind()
