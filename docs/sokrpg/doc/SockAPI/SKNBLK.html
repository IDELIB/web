SKNBLK

構文： 
     C*ノンブロックの設定------------------------------------*
     C                     CALL 'SKNBLK'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM '1'       FLG     1        1=ON 0=OFF
     C*------------------------------------------------------*
解説：
ソケットのブロック／ノンブロックを切り替える。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。
FLG　(入力)
    '1'でノンブロックの設定オン、'0'でオフ、'1' '2' ともに文字タイプです。

実行されるソケット関数：
ioctl()
