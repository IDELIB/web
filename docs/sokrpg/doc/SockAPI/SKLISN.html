SKLISN

構文： 
     C*ソケットリッスン--------------------------------------*
     C                     CALL 'SKLISN'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C*------------------------------------------------------*
解説：
ソケットへの接続要求を待機します。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。

実行されるソケット関数：
listen()
