SKSHUT

構文： 
     C*シャットダウン送受信禁止------------------------------*
     C                     CALL 'SKSHUT'
     C                     PARM            SD      4
     C                     PARM        	   RC      1
     C                     PARM 'B'   	   HOW     1
     C* ソケット送信禁止 ------------------------------------*    
     C                     CALL 'SKSHUT'                            
     C                     PARM           SD      4                 
     C                     PARM           RC      1                 
     C                     PARM 'S'       HOW     1                 
     C* ソケット受信禁止 ------------------------------------*    
     C                     CALL 'SKSHUT'                            
     C                     PARM           SD      4                 
     C                     PARM           RC      1                 
     C                     PARM 'R'       HOW     1                 
     C*------------------------------------------------------*
解説：
ソケットを遮断します。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、'E'に設定される。
HOW　(入力)
    送信，受信，送受信のどれを禁止するかを指示する
    • 'B' : 送受信の禁止
    • 'S' : 送信の禁止
    • 'R' : 受信の禁止

実行されるソケット関数：
shutdown()
