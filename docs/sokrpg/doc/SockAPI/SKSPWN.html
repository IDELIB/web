SKSPWN

構文： 
     C*子プロセス生成----------------------------------------*
     C                     CALL 'SKSPWN'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM           PID     4
     C                     PARM 'USRPGM'  PGM    10
     C                     PARM 'USRLIB'  LIB    10
     C*------------------------------------------------------*
解説：
ソケットをからデータを受信します。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号、起動される子プロセスの第一パラメーターとして渡される
RC　(出力)
    エラーが発生した場合に'E'に設定される。
PID　(出力)
    起動した子プロセスのプロセスID番号。
PGM　(入力)
    起動した子プロセスで実行するユーザープログラム名
LIB　(入力)
    起動した子プロセスで実行するユーザープログラムのライブラリー名（必須）

実行される関数：
spawn()
