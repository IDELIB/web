SKTAKE

構文： 
     C* ソケット獲得を待つ ------------------------------------*  
     C                     CALL 'SKTAKE'                          
     C                     PARM           SD      4               
     C                     PARM           RC      1               
     C                     PARM 'SOKBCH  'APLID   8               
     C*--------------------------------------------------------*
解説：
他のJOBより、SKGIVEを利用して、ソケットディスクリプターが渡されるのを待機します。

パラメーター：
SD　(出力)
    獲得したソケットディスクリプターの番号、以降の番号で通信出来る
RC　(出力)
    エラーが発生した場合に'E'に設定される。
APLID　(入力)
    ソケットディスクリプターの受渡しの可能な複数JOBを識別するために付ける任意の文字列

実行される関数：
takedescriptor()
