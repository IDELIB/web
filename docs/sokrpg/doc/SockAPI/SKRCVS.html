SKRCVS

構文： 
     C* DATA受信 --------------------------------------------*
     C                     CALL 'SKRCVS'
     C                     PARM           SD      4
     C                     PARM           RC      1
     C                     PARM '256'     BUFLEN  4
     C                     PARM           BUF
     C*------------------------------------------------------*
解説：
ソケットからデータを受信後、シフトJISからEBCDICにデータ変換する。

パラメーター：
SD　(入力)
    使用するソケットディスクリプターの番号
RC　(出力)
    エラーが発生した場合、および、通信相手がソケットをクローズした場合に'E'に設定される。
BUFLEN　(入力)
    受信するバッファーの長さ（９９９９まで可能）、戻りは、実際に受信し変換処理後のデータの長さ。 ゼロの戻り値は通信相手がソケットをクローズしたことを示す。
BUF　(入力)
    実際にデータを受信するバッファー。少なくともBUFLENで指定した長さは必要。

    漢字変換の際にシフトコードが追加される為、その分の長さをBUFLENよりも大きく用意することが必要

実行されるソケット関数：
recv()
iconv()
