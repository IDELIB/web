［例］

     H            Y/                                    1
     H*---------------------------------------------------------------*
     H* PROGRAM-ID    :    SOKRPGNM                                   *
     H* REMARKS       :    RPGホスト名検索                            *
     H* AUTHOR        :    Y.IDE                                      *
     H* DATE-WRITEN   :    1999/03/20                                 *
     H* VERSION       :    01.00 ORIGINAL                             *
     H*---------------------------------------------------------------*
     FSOKNMFM CF  E                    WORKSTN
     C           TAG#01    TAG
     C                     EXFMTSC01
     C           *IN03     IFEQ '1'
     C                     GOTO TAG#E
     C                     ENDIF
     C                     MOVE *BLANK    FLD02
     C                     MOVE *BLANK    FLD03
     C                     MOVELFLD01     HOSTNM
     C*ホスト名検索------------------------------------------*
     C                     CALL 'SKGETH'
     C                     PARM           HOSTNM128
     C                     PARM           RC      1
     C                     PARM           IPSTR  15
     C                     PARM           IPHOST128
     C*--------------------------------------------------------*
     C*
     C           RC        IFEQ 'E'
     C                     MOVEL'ERROR   'FLD02
     C                     GOTO TAG#01
     C                     ENDIF
     C*
     C                     MOVELIPSTR     FLD02
     C                     MOVELIPHOST    FLD03
     C                     GOTO TAG#01
     C           TAG#E     TAG
     C*
     C                     SETON                     LR
     C                     RETRN

【解説】

IPアドレスｏｒホスト名をパラメーターとして渡すと、IPアドレスとホスト名を戻す。

IPアドレスｏｒホスト名のいずれでも検索に失敗した場合は、RCに'E'が設定される。
